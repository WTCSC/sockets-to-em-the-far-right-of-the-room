Order of Operations of Game:
1. All the different clients will connect to the server, with the server sending their player numbers
2. The server starts the game when all the clients the people want are connected
3. The actual game loop
   a. The server gets and sends out the prompt and hands for all of the players
   b. Each client gets and sends the card that each person picks
   c. The server sends out the list of all of the different voting cards
   d. Each client sends out the card they voted for
   e. The server sends out the voting results
   f. It repeats
